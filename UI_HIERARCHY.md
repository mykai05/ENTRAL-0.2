# Command OS UI Hierarchy

Command OS extends the existing ENTRAL dashboard into three functional regions.

## Left Navigation

The left navigation organizes:

- Command overview
- Generals
- Operations
- Infrastructure
- Analytics
- Settings

It also shows a compact chain-of-command tree so created Soldiers remain discoverable.

## Center Graph

The center remains the existing full-screen 3D atomic/neural graph. ENTRAL sits at the center, Generals orbit as major command shells, and Soldiers/Operations cluster under their parent command.

## Right Command Console

The right console is persistent. It is the primary path for navigation, command execution, mock Soldier creation, mock Operation creation, status checks, logs, and AI conversation.

## Inspector

The node inspector overlays beside the graph and shows selected node details: hierarchy position, health, permissions, tools, children, logs, activity, and mock execution status.

## Responsiveness

On smaller screens, navigation collapses away first and the command console becomes the primary drawer-like surface while the graph remains usable.
